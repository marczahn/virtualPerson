# Virtual Person

This is a hobby project where I want to **simulate** a person with their biology and psychology. It is an ongoing experiment to provide more realistic NPCs in games on the long run. The two first versions lacked of either human needs or reflection.

This project is not in any way production ready. With the support of AI, in iterations, I want to bring the different aspects better and better together to have at the very end a plausible acting and reacting character.
